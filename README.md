# Deep Impact

The goal of this gem is handle deep stringifying of keys and hashes,
in arrays and hashes of unknown depth.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'deep_impact'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install deep_impact
